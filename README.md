# cvotercrypto.org

Portfolio website for an independent course specialist.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- Framer Motion
- shadcn/ui

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

## Deploy

The project is ready for any Next.js-compatible host (Vercel, Netlify, Render, Railway, etc.).
